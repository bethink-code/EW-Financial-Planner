# Project: Elite Wealth — Portfolio Dashboard Concepts

Read `BRIEF.md` in full before doing anything — it is the source of truth for this build.

- `BRIEF.md` — the complete build brief (context, three concepts, attention layer, Level 2 panel spec, demo data, visual tokens, acceptance checklist).
- `reference-prototype.html` — a working single-file prototype of all three concepts. Use it as the functional reference for layout, copy, and interactions wherever the brief is ambiguous. The brief wins where they differ.

Build rules:
- Static front-end only, no backend, no network-dependent assets (must run offline in a client demo).
- Match the Elite Wealth visual language defined in BRIEF.md §6 — this must look like their product evolved, not replaced.
- The attention/readiness layer is a parallel layer woven into every concept, never a standalone screen.
- Verify your work against the acceptance checklist in BRIEF.md §10 before finishing.
